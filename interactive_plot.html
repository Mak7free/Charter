Dashboard Guide

Welcome to the interactive cell visualizer. Here is how to use the controls:

  • Sphere Appearance: Adjusts the "cell membrane" bubble. Use "Sphere Size" to scale the reference boundary.
  • Data Points: Controls how large the data dots appear. If they overlap, reduce these values.
  • Readability: Key for export. "Label Spacing" pushes text labels further out to prevent overlapping.
  • Grid & Axes: Increase these font sizes if you plan to take screenshots for papers/presentations.
